# datathon_2026

My team's submission to the 2026 Brandeis Datathon, in which we try to find good betting strategies for March Madness with machine learning/sports analytics. We found that they can be contextually pretty useful for finding edges in betting, though championship-level prediction is still ruled by the market/smart money.

NOTE: The specific code on the script may have outdated file naming convention (ie: "datathon 2026 set" instead of "datathon_2026 set" etc)
